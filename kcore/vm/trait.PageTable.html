Create a new page table.

fn switch(&self)[src]

Switch to this page table.

SAFETY: All pages mapped before switching should be accessiable after that.

That means the implementation can batch the map and unmap requests.

fn map(&mut self, va: usize, pg: &Page<Self>) -> Result<(), Error>[src]

Map virtual address [va, va+pg.size()) to physical page pg.

It is required to be atomic, i.e., undo the mapping if failed. Caller guarantees that pages in [va, va+pg.size()) are unmapped.

fn unmap(&mut self, va: usize, len: usize)[src]

Unmap the pages in [va, va+len) in this page table.

Caller guarantees that [va, va+len) has been mapped before.
